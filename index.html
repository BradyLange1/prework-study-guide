Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: pulls the current main code down form the repo
  • cd directory: changes the directory that you are in
  • git add -A: stages all files in the working branch
  • git commit -m "message": commits all files in the working brach and allows for a message to be added
